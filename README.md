# Portal de Cidadania Digital e Inteligência Artificial 🌐

### Objetivo do Projeto
Este projeto foi desenvolvido como um portal de conscientização comunitária para capacitar internautas a identificarem mídias manipuladas por IA (deepfakes) e instruí-los sobre a importância da checagem estrutural de notícias para conter a proliferação da desinformação em massa na sociedade contemporânea.

### Tecnologias Utilizadas
* **HTML5 Semântico:** Uso correto de tags como `<main>`, `<section>`, `<form>`, `<button>`.
* **CSS3 Avançado:** Controle fino de box-modeling, sistema de layout estruturado em Flexbox, variáveis nativas para gerenciamento de temas e Media Queries para responsividade móvel.
* **JavaScript Puro (ES6):** Manipulação reativa do DOM, tratamento lógico de inputs e condicionais baseadas nas escolhas feitas pelo usuário.

### Comandos e Prompts de IA Utilizados no Processamento
1. *"Gere uma estrutura de documento HTML5 limpa e semântica com foco em um formulário validador de URL e um Quiz educativo."*
2. *"Crie regras CSS baseadas em variáveis de paleta tecnológica utilizando Flexbox para o alinhamento de cartões verticais."*
3. *"Escreva um script JavaScript estruturado que intercepte o comportamento padrão de formulários, calcule pontuações e renderize feedbacks dinâmicos."*
